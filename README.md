# crypto_rsi_checker
check RSI number for crypto from bybit exchange


#env and run
setup venv :
````
python -m venv venv
````

run venv :
````
venv\scripts\activate
````

install packages :
````
pip install -r requirements.txt
````

packages list :
````
ccxt , pandas , ta
````


for run use this:
````
python main.py
````
